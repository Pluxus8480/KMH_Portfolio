PBR (Physically Based Rendering) 재질 시스템

Part 1: PBR이란?

PBR이 기존 렌더링과 다른 점은 무엇인가요?

PBR은 물리 법칙에 기반한 렌더링입니다. 빛이 실제로 물질과 상호작용하는 방식을 시뮬레이션합니다.

기존 방식은 "이 정도 밝기가 예뻐 보이니까"였다면, PBR은 "에너지 보존 법칙에 따라 이렇게 되어야 해"입니다.

🎯 비유로 이해하기

기존 방식은 아티스트의 직관에 의존했습니다. 낮에는 이 값, 밤에는 저 값...

PBR은 물리 공식을 따릅니다. 어떤 조명에서도 일관되게 사실적으로 보입니다.

Part 2: 반사 방정식 (Rendering Equation)

PBR의 수학적 기반은 무엇인가요?

모든 것의 시작은 렌더링 방정식입니다:

Lo(p, ωo) = Le(p, ωo) + ∫Ω fr(p, ωi, ωo) Li(p, ωi) (n · ωi) dωi
Lo = 나가는 빛 (우리가 보는 것)
Le = 자체 발광
fr = BRDF (양방향 반사 분포 함수)
Li = 들어오는 빛
n · ωi = 코사인 항 (램버트 법칙)
Ω = 모든 입사 방향에 대한 적분

렌더링 방정식의 핵심

BRDF가 뭔가요?

BRDF(Bidirectional Reflectance Distribution Function)는 "빛이 어떻게 반사되는가"를 정의하는 함수입니다.

입사각과 반사각이 주어지면, 그 조합에서 얼마나 많은 빛이 반사되는지 알려줍니다.

Part 3: Cook-Torrance BRDF

실시간 그래픽에서 주로 쓰는 BRDF는?

Cook-Torrance BRDF가 업계 표준입니다. Diffuse와 Specular 두 부분으로 나뉩니다:

fr = kd × fLambert + ks × fCookTorrance
Diffuse (난반사)
fLambert = albedo / π
Specular (정반사)
fCook-Torrance = D × F × G / 4(ωo·n)(ωi·n)

D = Normal Distribution Function (미세면 분포)
F = Fresnel Equation (프레넬)
G = Geometry Function (기하 감쇠)

Cook-Torrance BRDF 구조

Part 4: D, F, G 함수

D, F, G 각각은 무엇을 의미하나요?

Specular 반사의 세 가지 핵심 요소입니다:

함수 역할 영향받는 파라미터
D (NDF) 하이라이트 모양 Roughness
F (Fresnel) 각도에 따른 반사량 F0 (기본 반사율)
G (Geometry) 미세면 가림 효과 Roughness
HLSL - PBR 핵심 함수들
// D: GGX/Trowbridge-Reitz NDF
float D_GGX(float NdotH, float roughness)
{
    float a = roughness * roughness;
    float a2 = a * a;
    float denom = NdotH * NdotH * (a2 - 1.0) + 1.0;
    return a2 / (PI * denom * denom);
}

// F: Fresnel-Schlick
float3 F_Schlick(float cosTheta, float3 F0)
{
    return F0 + (1.0 - F0) * pow(1.0 - cosTheta, 5.0);
}

// G: Smith's method with GGX
float G_Smith(float NdotV, float NdotL, float roughness)
{
    float r = roughness + 1.0;
    float k = (r * r) / 8.0;
    float ggx1 = NdotV / (NdotV * (1.0 - k) + k);
    float ggx2 = NdotL / (NdotL * (1.0 - k) + k);
    return ggx1 * ggx2;
}

Part 5: 핵심 파라미터

PBR 머티리얼의 주요 입력은 무엇인가요?

대부분의 PBR 워크플로우는 Metallic-Roughness 방식을 사용합니다:

파라미터 설명 값 범위
Base Color 표면의 기본 색상 (알베도) sRGB 색상
Metallic 금속인가 비금속인가 0 (비금속) ~ 1 (금속)
Roughness 표면 거칠기 0 (매끄러움) ~ 1 (거침)
Normal 표면 미세 요철 탄젠트 공간 법선
AO 환경 차폐 0 (완전 차폐) ~ 1 (노출)
Metallic이 중요한 이유는?

금속과 비금속은 빛과 완전히 다르게 상호작용합니다:

• 금속: Diffuse 없음, 색이 있는 Specular

• 비금속: 대부분 Diffuse, 무채색 Specular

Metallic은 0 또는 1만 사용! 중간값은 물리적으로 말이 안 됩니다. (예외: 먼지 낀 금속)

Part 6: 전체 셰이더 구현

PBR 셰이더의 전체 구조는?

모든 요소를 합쳐 최종 색상을 계산합니다:

HLSL - 단순화된 PBR Shader
float3 PBR(float3 albedo, float metallic, float roughness,
           float3 N, float3 V, float3 L, float3 lightColor)
{
    float3 H = normalize(V + L);

    float NdotL = max(dot(N, L), 0.0);
    float NdotV = max(dot(N, V), 0.0);
    float NdotH = max(dot(N, H), 0.0);
    float VdotH = max(dot(V, H), 0.0);

    // F0: 기본 반사율 (비금속 = 0.04, 금속 = albedo)
    float3 F0 = lerp(float3(0.04, 0.04, 0.04), albedo, metallic);

    // Cook-Torrance BRDF
    float  D = D_GGX(NdotH, roughness);
    float3 F = F_Schlick(VdotH, F0);
    float  G = G_Smith(NdotV, NdotL, roughness);

    float3 specular = (D * F * G) /
                      (4.0 * NdotV * NdotL + 0.0001);

    // Diffuse (금속은 없음)
    float3 kD = (1.0 - F) * (1.0 - metallic);
    float3 diffuse = kD * albedo / PI;

    return (diffuse + specular) * lightColor * NdotL;
}
Metallic = 0 (플라스틱)
Diffuse: 색상 있음
Specular: 흰색
F0 = 0.04 (4%)
Metallic = 1 (금)
Diffuse: 없음
Specular: 금색
F0 = albedo

📌 핵심 정리

  • PBR은 물리 법칙에 기반, 어떤 조명에서도 일관된 결과
  • Cook-Torrance BRDF = Diffuse (Lambert) + Specular (D×F×G)
  • D는 하이라이트 모양, F는 프레넬, G는 기하 감쇠
  • Metallic으로 금속/비금속 구분 (0 또는 1 사용)
  • Roughness로 표면 거칠기 조절 (하이라이트 크기)
