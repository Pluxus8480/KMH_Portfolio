컴파일러를 컴퓨터로 만든다?

템플릿 메타프로그래밍(TMP)으로 컴파일 타임 연산하기

Part 1: TMP란 무엇인가?

템플릿 메타프로그래밍이 뭔가요?

컴파일 타임에 템플릿을 사용해 계산이나 코드 생성을 수행하는 기법입니다.

🎯 비유로 이해하기

요리를 할 때, 재료를 미리 손질해두면 실제 조리 시간이 줄어듭니다.

TMP는 컴파일 타임에 "미리 계산"해서 런타임을 빠르게 합니다.

컴파일 타임 팩토리얼
// 런타임 팩토리얼
int factorial(int n) {
    return n <= 1 ? 1 : n * factorial(n - 1);
}

// 컴파일 타임 팩토리얼 (TMP)
template<int N>
struct Factorial {
    static constexpr int value = N * Factorial<N - 1>::value;
};

template<>
struct Factorial<0> {  // 재귀 종료 조건
    static constexpr int value = 1;
};

// 사용
constexpr int result = Factorial<5>::value;  // 컴파일 타임에 120 계산!

TMP의 결과는 컴파일 타임 상수로, 런타임 비용이 0입니다!

Part 2: 타입 조작

타입도 컴파일 타임에 조작할 수 있나요?
타입 조작 예시
// const 제거
template<typename T>
struct RemoveConst {
    using type = T;
};

template<typename T>
struct RemoveConst<const T> {
    using type = T;
};

// 사용
using A = RemoveConst<const int>::type;  // int

// 포인터 제거
template<typename T>
struct RemovePointer {
    using type = T;
};

template<typename T>
struct RemovePointer<T*> {
    using type = T;
};

using B = RemovePointer<int*>::type;  // int

// 조건부 타입 선택 (std::conditional)
template<bool Cond, typename T, typename F>
struct Conditional {
    using type = T;
};

template<typename T, typename F>
struct Conditional<false, T, F> {
    using type = F;
};

using C = Conditional<(sizeof(int) > 2), long, short>::type;  // long

Part 3: 컴파일 타임 조건문과 반복

컴파일 타임 조건문
// if constexpr (C++17) - 가장 현대적인 방법
template<typename T>
auto process(T value) {
    if constexpr (std::is_integral_v<T>) {
        return value * 2;  // 정수형
    } else if constexpr (std::is_floating_point_v<T>) {
        return value / 2.0;  // 부동소수점
    } else {
        return value;  // 그 외
    }
}

// 전통적인 TMP 방식 (SFINAE / 특수화)
template<typename T, typename Enable = void>
struct Processor;

template<typename T>
struct Processor<T, std::enable_if_t<std::is_integral_v<T>>> {
    static T process(T val) { return val * 2; }
};

template<typename T>
struct Processor<T, std::enable_if_t<std::is_floating_point_v<T>>> {
    static T process(T val) { return val / 2.0; }
};
컴파일 타임 반복 (재귀)
// N개 요소의 합
template<int... Ns>
struct Sum;

template<>
struct Sum<> {
    static constexpr int value = 0;
};

template<int N, int... Rest>
struct Sum<N, Rest...> {
    static constexpr int value = N + Sum<Rest...>::value;
};

constexpr int total = Sum<1, 2, 3, 4, 5>::value;  // 15

// Fold Expression (C++17)
template<int... Ns>
constexpr int sum() {
    return (Ns + ...);  // fold expression
}

constexpr int total2 = sum<1, 2, 3, 4, 5>();  // 15

Part 4: 가변 템플릿 (Variadic Templates)

여러 개의 템플릿 인자를 받으려면?
Variadic Templates
// 가변 인자 개수 세기
template<typename... Ts>
struct Count {
    static constexpr size_t value = sizeof...(Ts);
};

static_assert(Count<int, float, double>::value == 3);

// 타입 리스트
template<typename... Ts>
struct TypeList {};

// 첫 번째 타입 추출
template<typename List>
struct Front;

template<typename T, typename... Ts>
struct Front<TypeList<T, Ts...>> {
    using type = T;
};

using List = TypeList<int, float, double>;
using First = Front<List>::type;  // int

// 타입 포함 여부 확인
template<typename T, typename List>
struct Contains;

template<typename T>
struct Contains<T, TypeList<>> : std::false_type {};

template<typename T, typename... Ts>
struct Contains<T, TypeList<T, Ts...>> : std::true_type {};

template<typename T, typename U, typename... Ts>
struct Contains<T, TypeList<U, Ts...>> : Contains<T, TypeList<Ts...>> {};

static_assert(Contains<int, TypeList<int, float>>::value == true);
static_assert(Contains<double, TypeList<int, float>>::value == false);

Part 5: 실전 TMP 패턴

튜플 구현
// 간단한 튜플 구현
template<typename... Ts>
struct Tuple;

template<>
struct Tuple<> {};

template<typename T, typename... Ts>
struct Tuple<T, Ts...> {
    T value;
    Tuple<Ts...> rest;

    Tuple(T v, Ts... vs) : value(v), rest(vs...) {}
};

// 인덱스로 접근
template<size_t I, typename T>
struct TupleElement;

template<typename T, typename... Ts>
struct TupleElement<0, Tuple<T, Ts...>> {
    using type = T;
    static T& get(Tuple<T, Ts...>& t) {
        return t.value;
    }
};

template<size_t I, typename T, typename... Ts>
struct TupleElement<I, Tuple<T, Ts...>> {
    using type = typename TupleElement<I - 1, Tuple<Ts...>>::type;
    static auto& get(Tuple<T, Ts...>& t) {
        return TupleElement<I - 1, Tuple<Ts...>>::get(t.rest);
    }
};

template<size_t I, typename... Ts>
auto& get(Tuple<Ts...>& t) {
    return TupleElement<I, Tuple<Ts...>>::get(t);
}

// 사용
Tuple<int, float, double> t(1, 2.0f, 3.0);
int a = get<0>(t);      // 1
float b = get<1>(t);    // 2.0f
double c = get<2>(t);   // 3.0
컴파일 타임 문자열
// 컴파일 타임 문자열 해시 (FNV-1a)
constexpr uint32_t fnv1a_hash(const char* str, size_t len) {
    uint32_t hash = 2166136261u;
    for (size_t i = 0; i < len; i++) {
        hash ^= static_cast<uint32_t>(str[i]);
        hash *= 16777619u;
    }
    return hash;
}

constexpr uint32_t operator""_hash(const char* str, size_t len) {
    return fnv1a_hash(str, len);
}

// 사용: switch문에서 문자열 비교
void process(const std::string& cmd) {
    switch (fnv1a_hash(cmd.c_str(), cmd.size())) {
        case "start"_hash:
            start();
            break;
        case "stop"_hash:
            stop();
            break;
        case "pause"_hash:
            pause();
            break;
    }
}

Part 6: Modern C++에서의 TMP

최신 C++에서 TMP가 어떻게 바뀌었나요?
기능 전통적 TMP Modern C++
컴파일 타임 계산 재귀 템플릿 constexpr 함수
조건부 컴파일 SFINAE, 특수화 if constexpr (C++17)
타입 제약 enable_if Concepts (C++20)
가변 인자 처리 재귀 언팩 Fold Expressions
Modern C++ 스타일
// C++20 Concepts
template<typename T>
concept Numeric = std::is_arithmetic_v<T>;

template<Numeric T>
T square(T x) {
    return x * x;
}

// C++17 if constexpr
template<typename T>
auto stringify(T value) {
    if constexpr (std::is_same_v<T, std::string>) {
        return value;
    } else if constexpr (std::is_arithmetic_v<T>) {
        return std::to_string(value);
    } else {
        return std::string("unknown");
    }
}

// C++17 Fold Expressions
template<typename... Args>
void print(Args&&... args) {
    (std::cout << ... << args) << '\n';
}

print(1, " + ", 2, " = ", 3);  // "1 + 2 = 3"

Modern C++는 TMP를 더 읽기 쉽고 강력하게 만들었습니다!
