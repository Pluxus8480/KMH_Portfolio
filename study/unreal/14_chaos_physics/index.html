Chaos Physics 시스템

Part 1: Chaos란 무엇인가?

Chaos Physics가 기존 PhysX와 다른 점은?

Chaos는 언리얼 엔진 5에서 PhysX를 대체하는 자체 물리 엔진입니다. 특히 대규모 파괴(Destruction)에 최적화되어 있습니다.

항목 PhysX Chaos
개발사 NVIDIA Epic Games
파괴 시스템 APEX (제한적) Geometry Collection
대규모 시뮬레이션 성능 제한 최적화됨
언리얼 통합 서드파티 네이티브
🎯 비유로 이해하기

PhysX는 외주 건축가와 작업하는 것입니다.

Chaos는 사내 건축팀을 갖춘 것입니다. 더 긴밀한 협업과 커스터마이징이 가능합니다.

Part 2: Geometry Collection

파괴 가능한 오브젝트는 어떻게 만드나요?

Geometry Collection을 사용합니다. 메시를 프랙처(조각)로 분할한 에셋입니다.

Geometry Collection 생성 과정:

1. 원본 메시
   ┌───────────────────┐
   │                   │
   │    🏠 건물 메시   │
   │                   │
   └───────────────────┘
              │
              │ Fracture (분할)
              ▼
2. 프랙처된 메시
   ┌───────────────────┐
   │ ┌──┬──┬──┬──┬──┐ │
   │ │  │  │  │  │  │ │
   │ ├──┼──┼──┼──┼──┤ │
   │ │  │  │  │  │  │ │
   │ └──┴──┴──┴──┴──┘ │
   └───────────────────┘
              │
              │ 런타임 충격
              ▼
3. 물리 시뮬레이션
   ┌───────────────────┐
   │    💥 충격       │
   │  ┌──┐ ┌──┐       │
   │    ┌──┐    ┌──┐  │
   │ 조각들이 물리적으로│
   │ 분리되어 날아감    │
   └───────────────────┘

Geometry Collection 동작 흐름

Geometry Collection 생성 방법
// 에디터에서 생성
1. Content Browser → 우클릭
2. Physics → Geometry Collection
3. Static Mesh 선택

// 또는 Fracture Mode에서
1. Static Mesh Actor 선택
2. Fracture Mode 진입 (Shift+6)
3. Fracture 도구로 분할 패턴 적용
   • Uniform: 균일한 조각
   • Clustered: 군집 형태
   • Radial: 방사형
   • Planar: 평면 분할
4. Generate Geometry Collection

Part 3: Fracture 계층 구조

큰 조각이 먼저 떨어지고, 작은 조각으로 더 부서지게 하려면?

계층적 프랙처를 사용합니다. 충격이 강해질수록 더 작은 레벨로 분해됩니다.

계층적 프랙처 구조:

Level 0 (전체)
┌─────────────────────────────┐
│                             │
│                             │
│                             │
└─────────────────────────────┘
              │ 약한 충격
              ▼
Level 1 (큰 조각)
┌───────┬───────┬─────────────┐
│       │       │             │
│       │       │             │
└───────┴───────┴─────────────┘
              │ 강한 충격
              ▼
Level 2 (작은 조각)
┌──┬──┬──┬──┬──┬──┬──┬──┬──┬──┐
│  │  │  │  │  │  │  │  │  │  │
└──┴──┴──┴──┴──┴──┴──┴──┴──┴──┘

Damage Threshold가 높은 레벨일수록

계층적 파괴 구조

Fracture Mode에서 계층 생성
// 계층적 프랙처 워크플로우

1. 전체 메시 선택

2. Cluster Fracture 적용 (Level 1)

3. Level 1 조각들 선택

4. 추가 Fracture 적용 (Level 2)
   → Uniform or Radial
   → 더 작은 조각으로 분할

  Level 2: 500

Part 4: 파괴 트리거

어떻게 파괴를 발생시키나요?

물리 충돌, 데미지 시스템, 또는 직접 Field를 적용해서 파괴를 트리거합니다.

파괴 트리거 방법
// 방법 1: 물리 충돌
// 자동으로 충돌 강도에 따라 파괴

// 방법 2: Apply Damage
void AMyCharacter::DamageWall(AGeometryCollectionActor* Wall)
{
    // 위치 기반 데미지
    FVector HitLocation = GetHitLocation();
    float DamageAmount = 1500.0f;

    // Radial Damage 적용
    UGameplayStatics::ApplyRadialDamage(
        GetWorld(),
        DamageAmount,
        HitLocation,
        500.0f,  // Radius
        UDamageType::StaticClass(),
        TArray(),
        this
    );
}

// 방법 3: Field System
// 폭발, 스트레인 등의 물리 필드로 파괴

// Field System Actor에서
Radial Falloff Field
  → Magnitude: 10000
  → Position: 폭발 위치
  → Min/Max Range: 0~500
Field 유형 효과
External Strain 외부 힘으로 분리
Kill 즉시 파괴
Linear Velocity 조각에 속도 적용
Disable 물리 비활성화

Part 5: 성능 최적화

수백 개의 파괴 조각이 있으면 성능은 괜찮나요?

Chaos는 여러 최적화 기법을 제공합니다:

최적화 설명
Removal 시간 후 조각 자동 제거
Sleep 정지한 조각 시뮬레이션 중지
Distance Culling 먼 조각 숨김
Clustering 인접 조각 그룹화
Geometry Collection 최적화 설정
// Geometry Collection Component 설정

// 자동 제거
Removal:
  ☑ Enable Removal
  Removal Duration: 5.0  // 5초 후 제거

// 거리 기반 숨김
Cull Distance:
  Min: 0
  Max: 10000  // 이 거리 밖에서 숨김

// 시뮬레이션 조절
Physics:
  Max Level Set on Wake: 1  // 깨어날 때 최대 레벨
  Cluster Group Index: 0

// 충돌 설정
Collision:
  Collision Type: Particle
  Implicit Type: Box  // 간단한 콜리전

Part 6: Anchors와 Constraints

바닥에 붙어있다가 위만 부서지게 하려면?

Anchors를 사용해 특정 조각을 고정합니다. 기둥의 아래쪽은 고정되고 위쪽만 떨어지게 할 수 있습니다.

Anchor 사용 예시:

충격 전:
┌───────────────────┐
│                   │
│   기둥            │
│                   │
├───────────────────┤
│███ Anchor (고정) ███│
└───────────────────┘

충격 후:
       ┌───────┐
       │ 떨어진 │
       │ 조각들 │
       └───────┘
            │
            ▼ 낙하
├───────────────────┤
│███ 아래는 고정 ███│
└───────────────────┘

Anchor로 부분 고정

Anchor 설정
// Fracture Mode에서 Anchor 설정

1. Geometry Collection 선택
2. Fracture Mode → Cluster 탭
3. Auto Cluster 또는 Manual 선택
4. 고정할 조각 선택
5. Set Initial Dynamic State → Sleeping
6. Anchored 체크

// 또는 Proximity 기반 Anchor
Selection → Select by Proximity
  → 바닥 근처 조각만 선택
  → Anchored 설정

📌 핵심 정리

  • Chaos는 언리얼 5의 자체 물리 엔진으로 대규모 파괴에 최적화
  • Geometry Collection으로 메시를 프랙처 조각으로 분할
  • 계층적 프랙처로 충격 강도에 따른 단계적 파괴 구현
  • Field System으로 폭발, 힘 적용 등 파괴 트리거
  • Anchor로 특정 조각 고정, Removal로 자동 정리
