# 포트폴리오 GIF / 이미지 준비 체크리스트

세 프로젝트 페이지에 비어 있는 `img-placeholder` 21개의 작업 목록입니다.
한 번에 캡처할 수 있도록 게임별로 묶고, 각 항목마다 "무엇이 보여야 하는지" 와 "캡처 팁" 을 정리했습니다.

## 요약

| 게임 | 정지 이미지 | GIF | 합계 |
|---|---:|---:|---:|
| **Mullet MadJack** | 0 | 7 | 7 |
| **Ghost Runner** | 2 | 4 | 6 |
| **HI-Fi Rush** | 3 | 5 | 8 |
| **합계** | 5 | 16 | **21** |

## 공통 가이드

- **GIF 길이**: 3~6초 권장 (반복 재생 시 자연스러운 루프)
- **해상도**: 가로 800~1280px 정도 (페이지 컨테이너 max-width 가 900px)
- **파일 크기**: 가능하면 2~5MB 이하 (GIF 가 너무 크면 모바일에서 로딩 지연)
- **저장 위치**: `assets/images/projects/<game-name>/` 하위 폴더에 저장 권장
- **파일명 규칙**: 영문 케밥 케이스 (예: `boss-laser-pattern.gif`)
- **녹화 도구**: ShareX, ScreenToGif, OBS + GIF 변환

# 작업 후 다음 단계

GIF 들이 준비되면 각 페이지의 `img-placeholder` div 를 다음과 같이 교체합니다:

```html
<!-- Before -->
<div class="img-placeholder">
    <div class="img-placeholder__label">GIF: 원형 지뢰 링</div>
    플레이어 주변 360도로 깔리는 지뢰
</div>

<!-- After -->
<figure class="project-figure">
    <img src="../../assets/images/projects/mullet-madjack/boss-mine-ring.gif"
         alt="비홀더의 원형 지뢰 링 패턴 — 360도 30발 동시 폭발"
         loading="lazy">
    <figcaption>플레이어 주변 360도로 깔리는 지뢰</figcaption>
</figure>
```

> `project-figure` / `project-figure img` / `figcaption` 의 CSS 스타일은 `css/project-detail.css` 에 새로 추가가 필요합니다. 교체 작업 시 함께 진행하면 됩니다. GIF 가 다 준비되면 알려주세요 — 일괄 교체와 CSS 추가를 같이 처리하겠습니다.
