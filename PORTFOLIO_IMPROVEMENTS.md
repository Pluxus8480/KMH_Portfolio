# KMH Portfolio - 개선 제안서

현재 포트폴리오를 실전 채용/이직용으로 활용하기 위한 개선 항목 정리.

---

## 1. 서버 프로젝트 섹션 추가

**현재 문제**: Projects 섹션이 클라이언트(DirectX) 모작 4개뿐. 서버 역량이 전혀 드러나지 않음.

**제안**: 서버 관련 프로젝트를 추가하거나, 기존 프로젝트에 서버 구성 요소를 보여주는 항목 추가.

**예시 항목**:
- GNISoft MMORPG 라이브 서버 운영 경험 (NDA 범위 내에서)
- 개인 게임 서버 프로젝트 (IOCP 기반 채팅/매치메이킹 등)
- 네트워크 동기화 데모 (클라이언트+서버 통합 시연)

**작업 방식**: 프로젝트 내용을 알려주면 카드 형태로 추가 가능. 공개 가능한 서버 프로젝트가 있다면 GitHub 링크와 함께 추가.

---

## 2. Skills 섹션 - 서버 기술 스택 강화

**현재 문제**: Tech Stack에 서버 관련 기술이 Web/Backend에 Node.js, PHP 정도만 있음. 게임 서버 핵심 기술 누락.

**제안**: 게임 서버 전용 카드를 추가하거나 기존 카드 재구성.

**추가할 기술 예시**:
- IOCP / epoll
- TCP/UDP 소켓 프로그래밍
- 패킷 직렬화/역직렬화
- 게임 서버 아키텍처 (Dedicated Server)
- 동기화 전략 (State Sync, Lag Compensation)
- 부하 테스트 / 프로파일링

**필요한 정보**: 실제로 실무에서 사용하는 서버 기술 스택 목록을 알려주면 정확하게 반영 가능.

---

## 3. 프로젝트 상세 페이지

**현재 문제**: 각 프로젝트가 메인 페이지 카드 한 장에 모든 설명이 압축되어 있음. 기술적 깊이를 보여주기 어려움.

**제안**: 각 프로젝트별 상세 페이지 (study 콘텐츠처럼) 생성.

**상세 페이지 내용**:
- 아키텍처 다이어그램
- 핵심 기술 구현 설명 (렌더링 파이프라인, 물리 엔진 구조 등)
- Before/After 최적화 수치
- 스크린샷/GIF 갤러리
- 담당 역할 상세 설명 (팀 프로젝트의 경우)

**작업 방식**: 각 프로젝트의 기술적 세부 내용을 알려주면 study 콘텐츠와 동일한 포맷으로 상세 페이지 생성 가능.

---

## 4. 이력서(Resume) 다운로드 버튼

**현재 문제**: 포트폴리오를 본 채용 담당자가 이력서를 별도로 요청해야 함.

**제안**: Hero 섹션이나 Contact 섹션에 PDF 이력서 다운로드 버튼 추가.

**작업 방식**: PDF 파일을 `/assets/resume.pdf`에 넣어주면 다운로드 버튼 추가 가능.

---

## 5. Experience 타임라인 - 역할 상세화

**현재 문제**: GNISoft 경력이 한 줄로 축약. 현재 메인 직무인데 구체적인 성과가 드러나지 않음.

**제안**: 현 직장 경험을 구체적으로 보강.

**보강 예시**:
- 담당 게임명 (NDA 허용 범위)
- 구체적 기술 성과 (서버 안정성 향상 수치, 동접 규모 등)
- 대만 서버 런칭 관련 글로벌화 경험
- 라이브 서비스 장애 대응 경험

**필요한 정보**: 공개 가능한 범위에서 GNISoft 경력 상세 내용.

---

## 6. Study 섹션 홍보 강화

**현재 문제**: 117개 학습 콘텐츠가 있지만 메인 페이지에서 네비게이션 링크 하나로만 연결. 엄청난 자산이 잘 안 보임.

**제안**:
- 메인 페이지에 "Study" 미니 섹션 추가 (최근 학습 또는 카테고리별 콘텐츠 수 요약)
- "7개 카테고리, 117개 학습 노트" 같은 통계를 보여주는 배너

**작업 방식**: 바로 작업 가능. 원하는 위치와 스타일만 정해주면 됨.

---

## 7. 프로젝트 스크린샷/썸네일 이미지

**현재 문제**: 프로젝트 카드에 실제 게임 스크린샷 없이 이모지 placeholder만 있음.

**제안**: 각 프로젝트의 대표 스크린샷 또는 GIF를 썸네일로 추가.

**작업 방식**: 스크린샷 이미지 파일을 제공해주면 카드에 삽입 가능.

---

## 8. Open Graph / SEO 메타태그

**현재 문제**: SNS 공유 시 미리보기가 제대로 안 나옴 (og:image, og:description 등 미설정).

**제안**: 메타태그 추가로 LinkedIn/카카오톡 등에서 공유 시 깔끔한 미리보기.

**작업 방식**: 대표 이미지를 제공해주면 바로 추가 가능.
